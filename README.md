# 🌸 DevOps Portfolio Website

A modern, elegant, and visually stunning **DevOps Engineer Portfolio Website** built using **React + Vite + TypeScript + Tailwind CSS**.
Designed with soft pastel aesthetics, smooth animations, and a premium UI to showcase projects, skills, and experience.

---

## ✨ Features

* 🎨 **Elegant UI Design**

  * Soft pastel color palette (lavender, mint, blush tones)
  * Glassmorphism + subtle gradients
  * Section-based layout with visually distinct backgrounds

* ⚡ **Modern Frontend Stack**

  * React (with Vite)
  * TypeScript
  * Tailwind CSS

* 🎬 **Smooth Animations**

  * Scroll-based animations
  * Hover effects on cards (lift, glow, scale)
  * Micro-interactions for buttons and links

* 📂 **Projects Showcase**

  * Displays selected GitHub repositories
  * Extracts project details from README files
  * Each project includes:

    * Description
    * Tech stack
    * Direct GitHub link

* 🧠 **Professional Sections**

  * Hero (intro + tagline)
  * About Me
  * Skills (categorized)
  * Projects
  * Experience Timeline
  * Education
  * Certifications
  * Contact

* 🌙 **Bonus Features**

  * Dark/Light mode toggle
  * Smooth scrolling navigation
  * Loading skeletons for API data
  * Responsive design (mobile + desktop)

## 📦 Installation & Setup

```bash
# Clone the repository
git clone <your-repo-url>

# Navigate to project folder
cd <project-folder>

# Install dependencies
npm install

# Start development server
npm run dev
```
